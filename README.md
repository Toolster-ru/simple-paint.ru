#Онлайн рисовалка
Перед вами простая онлайн рисовалка. Просто выбирайте цвета и рисуйте курсором мыши. Чтобы скачать получившееся изображение, нажмите на кнопку "Скачать" в правом верхнем углу. Чтобы очистить холст, нажмите на кнопку "Очистить" в левом верхнем углу. Ссылка: [Онлайн рисовалка](http://simple-paint.ru)
